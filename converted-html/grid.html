Grid system

1. Add a class "page-container" to your application main wrapper 

<div id="application" class="page-container"></div>

2. Play with class "row" and "col1" ~ "col12" 

// Basic 2 column layout

<div id="application" class="page-container"></div>
  <div class="row">
    <div class="col6"></div>
    <div class="col6"></div>
  </div>
</div>


// Basic 3 column layout

<div id="application" class="page-container"></div>
  <div class="row">
    <div class="col4"></div>
    <div class="col4"></div>
    <div class="col4"></div>
  </div>
</div>


// Nested column layout

<div id="application" class="page-container"></div>
  <div class="row">
    <div class="col6">
      <div class="col6"></div>
      <div class="col6"></div>
    </div>
    <div class="col6">
      <div class="col6"></div>
      <div class="col6"></div>
    </div>
  </div>
</div>
