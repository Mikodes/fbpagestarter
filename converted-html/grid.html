810px Grid system

1. include 810pxGrid.css into your project. 

<link rel="stylesheet" type="text/css" href="PATH/TO/810pxGrid.css">

2. Add a class "page-container" to your application main wrapper 

<div id="application" class="page-container"></div>

3. Play with class "row" and "col1" ~ "col12" 

<div id="application" class="page-container">
  // Basic 2 column layout
  <div class="row">
    <div class="col6"></div>
    <div class="col6"></div>
  </div>

  // Basic 3 column layout
  <div class="row">
    <div class="col4"></div>
    <div class="col4"></div>
    <div class="col4"></div>
  </div>

  // Nested column layout
  <div class="row">
    <div class="col6">
      <div class="col6"></div>
      <div class="col6"></div>
    </div>
    <div class="col6">
      <div class="col6"></div>
      <div class="col6"></div>
    </div>
  </div>
</div>
