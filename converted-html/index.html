Fbpagestarter is front-end kit that makes it easy to create an application on Facebook.

Features

  • Loading and initialization Facebook's JS SDK
  • Facebook login
  • Dialogs (Add page tab, Feed, OAuth, Requests, Send)
  • Canvas Methods (setSize, setAutoGrow, scrollTo)
  • Check user's like status
  • 810px Grid system
