Trigger a facebook dialog is very simple, add a 'fb-dialog' class on your element, than add a 'data-fbactiontype' attribute with actions that you want to use.

Add Page Tab

The add page tab dialog allows a user to add an application to a Facebook Page which they administer.

Use data-fbactiontype = "add_page_tab"

<button class="fb-dialog" data-fbactiontype="add_page_tab">Add tab to a page</button>

Authentication

The authentication dialog allows a user to authorize an application as part of an authentication flow.

Use data-fbactiontype = "permissions", than add "data-permissions" attribute to control permissions

<button class="fb-dialog" data-fbactiontype="permissions" data-permissions="email,user_birthday,publish_actions">Ask Permissions</button>

Post to wall

The feed dialog allows a user to post a story to their Timeline

Use data-fbactiontype = "share", than add "data-link", "data-picture", "data-title", "data-caption", "data-description" attributes to control your message contents

<button class="fb-dialog" 
        data-fbactiontype="share"
        data-links="YOUR_LINK"
        data-picture="AN_IMAGE_AT_LEAST_200PX_WITH_ABSOLUTE_URL"
        data-title="YOUR_TITLE"
        data-caption="YOUR_CAPTION"
        data-description="YOUR_DESCRIPTION">
Ask Permissions</button>

*All these data attributes are required, url of data-link must be an absolute url, and I recommend a 300px*300px image for data-picture.

Multi Friend Selector Requests

The requests dialog allows a user to send a request to one or more of their friends

Use data-fbactiontype = "requests", than add "data-message" attribute to control your message contents

<button class="fb-dialog" data-fbactiontype="requests" data-message="REQUEST_MESSAGE">Multi Friend Selector Requests</button>

*The icon who display on requests dialog is the Facebook application icon.

Send

The Send Dialog allows a user to send a Facebook Message to one or more of their friends.

<button class="fb-dialog" 
        data-fbactiontype="send"
        data-name="TITLE"
        data-link="YOUR_LINK"
        data-picture="AN_IMAGE_AT_LEAST_200PX_WITH_ABSOLUTE_URL">
Multi Friend Selector Requests</button>

Scroll to
