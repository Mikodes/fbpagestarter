Triggering a facebook dialog is very simple. Simply add a 'fb-dialog' class on your element, then add a 'data-fbactiontype' attribute with actions that you want to use.

Add Page Tab

The add page tab dialog allows a user to add an application to a Facebook Page which they administer.

Use data-fbactiontype = "add_page_tab"

<button class="fb-dialog" data-fbactiontype="add_page_tab">Add tab to a page</button>

Authentication

The authentication dialog allows a user to authorize an application as part of an authentication flow.

Use data-fbactiontype = "permissions", than add "data-permissions" attribute to control permissions

<button class="fb-dialog" data-fbactiontype="permissions" data-permissions="email,user_birthday">Ask Permissions</button>

Post to wall

The feed dialog allows a user to post a story to their Timeline

Use data-fbactiontype = "share", than add "data-link", "data-picture", "data-title", "data-caption", "data-description" attributes to control your message contents

<button class="fb-dialog" 
        data-fbactiontype="share"
        data-links="YOUR_LINK"
        data-picture="AN_IMAGE_AT_LEAST_200PX_WITH_ABSOLUTE_URL"
        data-title="YOUR_TITLE"
        data-caption="YOUR_CAPTION"
        data-description="YOUR_DESCRIPTION">
Ask Permissions</button>

*All these data attributes are required. URL of data-link must be an absolute url - I recommend a 300px*300px image for data-picture.

Multi Friend Selector Requests

The request dialog allows a user to send a request to one or more of their friends

Use data-fbactiontype = "requests", than add "data-message" attribute to control your message contents

<button class="fb-dialog" data-fbactiontype="requests" data-message="REQUEST_MESSAGE">Multi Friend Selector Requests</button>

*The icon who display on requests dialog is the Facebook application icon.

Send

The Send Dialog allows a user to send a Facebook Message to one or more of their friends.

<button class="fb-dialog" 
        data-fbactiontype="send"
        data-name="TITLE"
        data-link="YOUR_LINK"
        data-picture="AN_IMAGE_AT_LEAST_200PX_WITH_ABSOLUTE_URL">
Multi Friend Selector Requests</button>

Scroll to

To scroll to a specific location of your canvas page.

Use 'fb-scrollto' class to activate this function, then add 'data-offset' to define a y-position as a scroll destination and use 'data-speed' value to control animation duration (default speed is 500).

<button class="fb-scrollto" data-offset="20" data-speed="500">Scroll to</button>

You can also call scrollto function out of the plugin with '$.fn.fbStarter.externalScrollto("OFFSET","SPEED")'.

<button id="custome-scrollto">Scroll to</button>
$(document).on('fb:initialized', function() {
  $("#custome-scrollto").on("click", function(e){
    e.preventDefault();
    $.fn.fbStarter.externalScrollto(50,500);
  })
});

Set Canvas Height

*This function isn't operable in this demo. It must set the 'autoResize' option to false in order to deactivate plugin's default autoresize function.

Tells Facebook to resize your canvas page to a specific height.

Use 'fb-setsize' class to activate this function, then add 'data-height' to define a new canavas height.

<button class="fb-setsize" data-height="300">Resize canvas height</button>

Auto-Resize Canvas Height

By default, there is already an autoresize function who extrude after canvas page loaded. If you add or remove parts of the dom and you want to autoresize the canvas page again, you can use this function: '$.fn.fbStarter.externalAutoresize()'.

$(document).on('fb:initialized', function() {
  $.fn.fbStarter.externalAutoresize();
});
